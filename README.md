# Shady Webshop

School exam project. Built the core for a working webshop with .Net and EPIServer.

## Installation

Download and install the following:

```
- Clone the project
- SQL Studio Manager
- Visual Studio
- SQL Server
- IIS
```

Install and run:

```
- npm install
- nuget packages
- A database
```

Check:

```
Startup.cs and Web.config for personal settings
```

## Run

```
Ctrl + F5
```
